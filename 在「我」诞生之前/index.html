在「我」诞生之前

我进入大学之前，并没有系统地学过编程，只凭着比较多的玩电脑的时间略有涉足。

本来还觉得当时自己有不少小玩意儿可以拿出来讲讲，但是现在回过头来看真的感觉不值一提。我在这里仅提及两个我最怀念的项目吧！

1. 植物大战僵尸战略版

这个项目我很早就开始做了（大约 2014 年），那时 @囧丫乙 做的 pvz js 版在贴吧大火，我一看，感觉非常好魔改。在此之前我已经做了很多 pvz 的自制 pak 了，所以基本上还是没什么难度的。

伴随着这个项目，我逐渐熟练地学会了 flash 的绘图和动画制作，虽然 flash 现在要死了（bushi），但是做出来的东西效果还是非常好的。

渐渐的就学会了一些基本的 js 语法，也可以对原来的植物、僵尸类做点修改，虽然被 @PAK向日葵 吐槽是“一个大喷菇模型走天下”，但是我觉得一般玩家看不出来hhhh

后来，当初一起弄 pvz 技术的网友们成立了江南游戏开发社（网络群组），挨炮、宅人康等网友对本项目帮助很大。

高中之后就没再管这个项目了，因此最后更新是 2016 年。由于也没有做项目文档，所以没有再继续下去，同时 chrome 对 gif 的支持还是有点缺陷，囧阿姨正在做新内核的 jspvz，试图使用 js 实现动画的实时渲染，而不是使用 gif，到目前为止已经工作的比较好了，但是还没有公开测试。

下面是一些游戏界面截图。我这里应该还有游戏的源码，最初的源码得到 @囧丫乙 的认可，挂载在她的服务器上，后来国家对在线网游的政策收紧，她的域名一再变动，现在我应该没有在线游玩的链接可以提供给您了。

2. “贴吧大神” 名片生成器

“贴吧大神” 是百度贴吧推出的一个内容产出用户的头衔，一些持续产出高质量内容的贴吧用户可以接受这个头衔，并且在这个组织的激励下持续产出更多的优质内容。

当时我做了不少植物大战僵尸的攻略，因此也在这个圈子里玩了很久。

“贴吧大神” 发帖需要带上组织统一制作的 “名片” 作为个性签名，一开始人还不多的时候，每一批新用户成为 “大神” 时官方就会很快做好名片发下来，但是随着后面人越来越多，官方似乎也有点懈怠，所以我与 @PAK向日葵 一起做了这个 Web 纯前端的名片生成器。

因为当时也是挂在 @囧丫乙 提供的服务器上，这里也没有更多链接可以提供了。
